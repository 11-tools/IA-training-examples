Relative vs Absolute Text Text Size for Resizing Text

The following page should be viewed in a browser that allows the user to resize only the text and not magnify the entire page (e.g., Firefox [View -> Zoom-> Zoom Text Only]).

If the entire page can be magnified with no content obscured using the full page zoom feature of web browsers then in meets WCAG 2.0 AA Success Criteria 1.4.4.

This is simply an example of the benefits of using relative units.

Absolute Text Size

Level One Heading

The contens of this container are sized using absolute units with a text font size of 20px and a H1 font size of 25px;. The container has an absolute width of 480px and a height of 240px.

Relative Text Size

Level One Heading

The contens of this container are sized using relative units with a text font size of 1.25em which equals 20px and a H1 font size of 1.25em which equals 25px;. The container has a relative width of 30em (480px) and a height of 15em (240px).
